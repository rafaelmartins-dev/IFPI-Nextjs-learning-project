# IFPI-Nextjs-learning-project

Projeto prático de estudo baseado na documentação oficial do Next.js (App Router). Construindo um Dashboard Financeiro completo para ir de iniciante a expert. O projeto cobre Autenticação, páginas protegidas, integração com banco de dados e funcionalidade CRUD (adição, edição e exclusão) de faturas.
